# 当たり待ち

当たるまで待つだけの確率シミュレーション。当選数と抽選総数を入力してスタートすると、当たりが出るまで自動で抽選し続けます。

## 技術スタック

- [Next.js](https://nextjs.org/) 16
- [React](https://react.dev/) 18
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## 開発環境

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) で確認できます。

## ビルド

```bash
npm run build
npm run start
```

## デプロイ

[Vercel](https://vercel.com/) でホスティングしています。`main` ブランチへのプッシュで自動デプロイされます。
